# utools-write

## 介绍

这是一个utools插件,化繁去简,支持快速的文本输入,没有其他任何干扰因素
2022-05-08: utools官方觉得功能太简单,拒绝收录市场,待后期丰富下功能😭

## 技术选型

插件基于Vue3+Vite2+NaiveUI

## 初始化配置

```sh
npm install
```

### 启动开发环境

```sh
npm run dev
```

### 打包编译

```sh
npm run build
```

### 注意

如果报错提示chalk模块未找到,执行下面命令手动安装

```shell
npm instal chalk
```
